Projects

Game of Words

It is a word-based game developed as part of a course in Object Oriented Programming (CSE110). It incorporates fundamental concepts of object-oriented design, file handling, and graphical user interface development using Java Swing. The game challenges players to guess words from given set words and varying lengths, across three difficulty levels Easy, Medium, and Hard while penalizing incorrect guesses.

Tic-Tac-Toe

It is a two player game where each player uses a signature. The player who successfully places three respective signatures in a vertical, horizontal, or diagonal row is the winner.

Railway Tickets Booking System

Developed a Railway Tickets Booking System as part of the CSE246: Algorithms course. The system utilizes advanced algorithms such as the KMP algorithm for efficient pattern matching to verify user uniqueness and Dijkstra's algorithm for optimal route planning. This project involved designing user registration, ticket booking, and fare calculation functionalities. It enhanced my understanding of algorithmic applications in solving real world problems and demonstrated the importance of efficiency in transportation systems.
